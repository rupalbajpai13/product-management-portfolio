

 Epic 1: AI Invoice Investigation
 ****************************************************
 User Story 1: US-001: Analyze Blocked Invoice
As an Accounts Payable Specialist, I want the AI assistant to analyze a blocked invoice and identify the most likely root cause, So that I can quickly understand the reason for the exception.

User Story 2 : US-002: Handle AI Uncertainty
As an Accounts Payable Specialist,I want the AI assistant to clearly indicate when it cannot determine a reliable root cause,
So that I can avoid acting on incorrect recommendations.
